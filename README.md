# Directory monitor

## Installation

```bash
$ git clone https://github.com/ghadd/edu-DirMonitor
$ cd edu-DirMonitor
$ mkdir build && cd build
$ cmake ..
$ cmake --build .
$ ./edu-DirMonitor

window pops up
```

## Running Tests

```bash
$ cd test
$ mkdir build && cd build
$ cmake ..
$ cmake --build .
$ ./edu-DirMonitor-Test

Running cxxtest tests (2 tests)..OK!
```
